# Comunio Mundial 2026 🏆⚽

App web progresiva (PWA) para seguir el Mundial 2026 con el Comunio.

## Cómo subir a GitHub Pages

### 1. Crear repositorio
- Ve a [github.com](https://github.com) e inicia sesión
- Pulsa el botón **"New"** (o el **"+"** arriba a la derecha → New repository)
- Nombre del repositorio: `mundial2026` (o el que quieras)
- Déjalo en **Public**
- Pulsa **"Create repository"**

### 2. Subir los archivos
- En la página del repositorio recién creado, pulsa **"uploading an existing file"**
- Arrastra TODOS los archivos de esta carpeta al área de subida:
  - `index.html`
  - `style.css`
  - `app.js`
  - `data.js`
  - `sw.js`
  - `manifest.json`
  - `icon-192.png`
  - `icon-512.png`
- Pulsa **"Commit changes"**

### 3. Activar GitHub Pages
- Ve a **Settings** del repositorio (pestaña arriba)
- En el menú izquierdo, pulsa **Pages**
- En "Branch" selecciona **main** y carpeta **/ (root)**
- Pulsa **Save**
- Espera 1-2 minutos

### 4. Tu URL
Tu app estará disponible en:
`https://TU_USUARIO.github.io/mundial2026/`

### 5. Instalar en Android
- Abre la URL en Chrome en tu móvil
- Aparecerá un banner "Añadir a pantalla de inicio" — pulsa **Añadir**
- Si no aparece el banner: menú de Chrome (⋮) → **"Añadir a pantalla de inicio"**

---

La app guarda los resultados automáticamente en el dispositivo y funciona sin internet una vez instalada.
